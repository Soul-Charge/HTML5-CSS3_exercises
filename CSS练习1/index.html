学习指南

亲爱的同学，欢迎你光临《Web应用程序设计（ASP.NET）》精品课程网站！从你踏入网站的第一刻起，我们有理由相信你一定对.NET平台的Web开发感兴趣，你一定想掌握这门神奇的技术，如果是这样，请耐心地阅读这段指南，它将会使你少走不少弯路！

一、网页设计基础

如果你对XHTML、CSS不了解或者了解得不多，那么建议你先从HTML学起，在这一部分你将熟练掌握HTML标签、DIV+CSS流式布局的技术。关于这部分内容请参考: HTML语言基础、CSS布局

如果你本身已经是一个非常优秀的网页界面设计师了，好可以跳过这一部分直接往下阅读。不过经典的文章还是值得推荐去读一读的，希望不要错过！

二、面向对象程序设计基础

进行.NET平台的Web开发，掌握一门面向对象程序设计语言是必须的！.NET平台支持多种程序设计语言，当然，目前大多数人习惯使用C#作为后台语言进行开发。因此，如果你对C#了解不多，建议你学习使用C#编写程序。关于这部分内容，请参考：C#程序设计

三、数据库

进行动态Web程序设计，数据库是必不可少的。因此你必须掌握一定的数据库知识，包括表、视图、SQL查询语言、存储过程等。关于这部分内容，请参考：SQL自学通

四、ASP.NET开发技术

恭喜你！终于要接触到.NET平台的Web开发技术了！.NET平台的Web开发涉及的知识面较广，入门难度不高，但精通需要一定的项目实战。本站提供了相关的参考资源，请参考Programming.ASP.NET.中文版、ASP.NET.2.0.数据库入门经典(第4版)

在学习ASP.NET的过程中，学习视频库会是你的好帮手，强烈推荐每一位学习ASP.NET的同学去观看！这里包含了96集学习视频，绝对是自学的好帮手！

五、项目实战

Web开发，重要的是动手！站在岸上是永远也学不会游泳的，所以，在掌握了相应的基础知识后，请拿一个实际的项目进行练手吧！本站提供了丰富的实战案例供你选择，详情请参考：课程案例库、课程考核选题

六、素材下载

在你进行设计与开发的过程中，也许会因为缺少相关的素材而烦恼！本站为你提供了丰富素材：模板库中提供了丰富的优秀网页模板；图库中包含了设计过程中经常用到的图标、图片等; 项目库中提供了丰富的成型项目供你参考；开发过程中需要多种工具，工具库为你提供了下载链接！
